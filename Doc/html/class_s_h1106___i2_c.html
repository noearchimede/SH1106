Detailed Description

I2C interface implementation class This class allows to communicate with an SH1106 through i2c interface.

This class provides low level i2c functions specific to the ATmega328 microcontroller (like init() and start()) as well as higher level functions that match SH1106 requirements (e.g. sendCommand() and controlByte()).

The public functions are defined in the SH1106_interface abstract class.
