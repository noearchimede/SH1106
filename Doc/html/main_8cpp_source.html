226 The print function allows to write ASCII printable characters (i.e. ASCII
227 characters 0x20 to 0x7E) as well as some special characters, obtained by
228 writing the following two character sequences.
229 The backslash '\' is used as escape character. Unless the user uses raw string
230 literals, the escape character must be written twice to pass it to the class
231 as a single backslash is interpreted by the compiler: e.g. write "\\^" to get
232 the escape sequence "\^" (up arrow).
233 
234 | ASCII control characters ||
235 |------|--------------------------------------|

249 |------|-------------|

252 | -> | RIGHT |
253 | -v | DOWN RIGHT |
254 | \v | DOWN |
255 | v- | DOWN LEFT |
256 | <- | LEFT |

258 
259 _Note: up sign is a circumflex accent, down is a lower case 'v'_

261 | Digits at exponent position ||
262 |--------------------------------------------------------------------|
