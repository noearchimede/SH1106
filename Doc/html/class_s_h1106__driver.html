Detailed Description

SH1106 OLED controller communication management.

Header of the SH1106_driver class

This class gives easy access to all SH1106 basic commands and provides an initialization and a write functions.

Constructor.

Note
This constructor must be used if the display is connected to the microcontroller via the I2C interface.
Parameters
widthWidth of the OLED screen (may be smaller than the 132 bytes RAM of the SH1106)
pagesNumber of pages shown on screen. A page is 8 lines. The RAM has 8 pages.
horizontalOffsetRAM address of the first column. It may be not 0 when the screen is less wide than the RAM (e.g. is sometimes 2 for 128 pixel screens).
i2cAddressaddress of the display on the i2c bus. defaults to 0x78.
i2cUseInternalPullupsset whether to use internal or external pullup resistors fot the i2c bus
